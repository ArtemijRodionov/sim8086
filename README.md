# sim8086
Decoder and interpreter for basic set of 8086 assembly instructions

## Build
`make all`

## Tests
1. Download assembly source code from https://github.com/cmuratori/computer_enhance/tree/main/perfaware/part1
2. Put source code to data/
3. Run `make test`
